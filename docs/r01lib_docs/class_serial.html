Detailed Description

Mbed-compatible UART class backed by interrupt-driven ring buffers.

Instantiate with a TX pin, RX pin, and optional baud rate. The constructor resolves the hardware peripheral, configures clocks, mux, and enables the LPUART. The destructor cleanly disables interrupts and de-initialises the peripheral.

◆ attach()

void Serial::attach ( func_ptr callback,
IrqType type = RxIrq )

Register a callback for RX or TX interrupts.

Attaching an RxIrq callback enables the LPUART receive interrupt and switches getc / readable to ring-buffer mode. Attaching a TxIrq callback stores a function that is called once the TX ring buffer has been fully drained.

Pass nullptr to remove a previously registered callback.

Parameters
callbackFunction to call on the selected interrupt event.
typeRxIrq (default) or TxIrq.

◆ getc()

int Serial::getc ( void )

Read one character.

If an RX callback has been attached, reads from the RX ring buffer and returns -1 if no data is available. If no callback is attached, blocks until a byte is received via LPUART_ReadBlocking.

Returns
Received byte as int (0–255), or -1 if the ring buffer is empty and interrupt-driven mode is active.
